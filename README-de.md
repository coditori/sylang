#### - (Liste der cSigns)

```
:>			Definition
@>			Autor, Schöpfer
#>			freigegeben
<<			erforderlich, Voraussetzung
$>			Alternative
+>			Vorteil
->			Nachteil
?>			Problem
==			Lösung
<>			im Gegensatz, gegenüber
=>			Punkt, Spitze
-			list (füge einen weiteren Bindestrich "-" hinzu für eine verschachtelte Ebene wie "-)
!>			Beispiel
~>			falsches Beispiel
>>			Verknüpfung
>>>			Referenz
nX			Nummerierte (setzen Sie eine Zahl vor cSigns wie "1>>" oder "1?>")
```

#### - (List of cSigns)

```
:>			definition
@>			author, creator
#>			released
<<			required, prerequisite
$>			alternative
+>			advantage
->			disadvantage
?>			problem
==			solution
<>			unlike, opposite
=>			point, tip
-			list (add another dash "-" for a nested level like "--)
!>			example
~>			wrong example
>>			link
>>>			reference
nX			Numbered ones (put a number before cSigns like "1>>" or "1?>")
```

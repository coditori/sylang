#### - (List of Code Signs)

```
:>			definition
@>			author, creator
#>			released
<<			required, prerequisite
$>			alternative
+>			advantage
->			disadvantage
?>			problem
==			solution
<>			unlike, oppsite
=>			point, tip
-			list (add another dash "-" for nested level like "--)
!>			example
~>			wrong example
>>			link
>>>			source
NcSigns			Numbered ones (put a number before cSigns like "1>>" or "1?>")
```
